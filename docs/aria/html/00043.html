aria_thermostat

This application showcases high performance graphics in a single buffer arrangement. The demonstration launches with a splash screen highlighting basic motion capability supported by graphics library. When running the application, the user can interface with it via capacitive single-fingered touch. Widgets such as circular slider and arc draw are used to create a modern thermostat UI. An information screen is used to explain the features demonstrated in this application. The application goes into automatic thermostat after 10 seconds of idle time (no touch input received).
