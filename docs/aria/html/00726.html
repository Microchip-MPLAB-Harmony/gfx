Aria Applications

Examples and demonstrations using Aria graphics library.

Topics
Name 
Description 
aria_quickstart 
This demonstration provides a touch-enabled starting point for the Aria Graphics.
 

 

 

aria_quickstart_ext_res 
This demonstration provides a touch-enabled starting point for reading images, string, and multiple fonts from both internal and external non-volatile memory (NVM).
 

 

 

aria_oven_controller 
This demonstration provides a practical oven UI application using the aria graphics library.

 

aria_radial_menu 
The application demonstrates four different uses of the radial menu widget.

 

aria_showcase 
The application demonstrates a subset of capabilities offered by the aria graphics library using Low-Cost Controllerless features with touch screen capabilities.

 

aria_showcase_reloaded 
The application demonstrates the circular and graphing widgets: arc, circular slider, circular gauge, pie chart, bar graph, line graph. These widgets are available and ready-to-use using the designer.

 

aria_thermostat 
The application demonstrates advanced graphics capabilities of graphics library.

 

aria_weather_forecast 
The application demonstrates a practical single-layered single-buffered application using the aria graphics library.

 

aria_mxt_configure 
The application demonstrates maXTouch configuration file uploading for the maXTouch Controller.

 

aria_infotainment 
The application demonstrates advanced graphics capabilities of aria graphics library.

 

aria_image_viewer 
This application demonstrates the use of blit features of the graphics processor using large image files.

 

aria_flash 
The aria_flash demonstration application serves as an external memory programmer to flash the off-chip non-volatile memory (NVM) SQI-Flash with the resources held on an Mass Storage Device (MSD), such as a USB MSD or a SD card, which can then be accessed by other applications, saving on-chip memory for other programs and resources.
 

 

 

aria_dashboard 
This application demonstrates a simulated vehicle dashboard running on the SAM A5D2 MPU built using the aria graphics library. This application is for graphics demonstration purposes only and is not designed to work with safety-critical systems.

 

aria_adventure 
The application demonstrates advanced graphics techniques such as parallax and sprite animation.

 

aria_benchmark 
This application demonstrates frame update rate metrics on the various rendering operations in the aria graphics library.
