Detailed Description

Legato widget definitions.

Defines the widget that are used in the UI library. Widgets are created and stored for later processing during a library context's update loop.

Typedef Documentation

◆ leBackgroundType

typedef enum leBackgroundType leBackgroundType

Used to define widget background types.

Specifies the different background types used for the widgets in the library. None - No background fill. Widget must defer to its parent to erase dirty pixels. This may cause additional overhead as clean pixels may be repainted as well. Fill - a scheme color is used to fill the widget rectangle. Cache - a local framebuffer cache is maintained by the widget and used to clean up dirty pixels. Will not cause a parent repaint event but will use additional memory to contain the cache.
