101 // *****************************************************************************
102 /* Structure:
103  leDrawSurfaceWidget
104 
105  Summary:
106  Implementation of a Drawsurface widget.
107 
108  Description:
109  A draw surface widget is a special widget that allows an application to
110  perform custom HAL draw calls during Legato's paint loop. To use, create
111  and add a draw surface widget to the desired place in the widget tree.
112  Then register for the callback through the API
113  'leDrawSurfaceWidget_SetDrawCallback'. This callback occurs during the
114  paint loop. The application should then be free to adjust the HAL
115  draw state and issue draw calls as desired. The HAL layer, buffer, or
116  context state must not be adjusted in any way.
117 
118  It is also important to not stall for too long during the draw callback.
119 
120  Remarks:
121  None.
122 */
123
